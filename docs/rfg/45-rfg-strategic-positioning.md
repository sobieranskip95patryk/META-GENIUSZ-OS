## Executive Summary

Rocket Fuel Girls (RFG) to **vertical creator brand** dla twórczyń wizualnych — modeli, influencerek, fotomodelki, model fitness — w ekosystemie META-GENIUSZ OS. RFG NIE jest modelem OnlyFans (zero adult content). RFG jest **professional portfolio + discovery platform** łączącą twórczynie z markami fashion, beauty, photography. RFG uruchamiany jako **drugi vertical po HHU** (v0.5+), korzystając z shared infrastructure (auth, payments, AI tools).

## Segment Definition

### Kim jest RFG Creator

| Atrybut | Wartość |
|---------|---------|
| **Profil** | Kobieta 18-35; modelka, influencerka, fotomodelka, content creatorka wizualna |
| **Lokalizacja MVP** | Polska (analogicznie do HHU: doc 40) |
| **Profesja** | Semi-professional → professional; portfolio online to must-have |
| **Current tools** | Instagram (główne), TikTok, Model Mayhem,PortfolioBox, Casting DB |
| **Pain points** | Brak dedykowanej platformy; IG algorithm ≠ portfolio; brak protection (fake castings, scam); trudno znaleźć legit brands |

### Kim NIE jest RFG Creator

| NIE jest | Powód |
|---------|-------|
| ❌ Adult content creator | RFG to professional platform; zero tolerance NSFW |
| ❌ Random IG influencer (lifestyle, beauty tips) | Niche za szeroki; RFG = visual creator, nie vlogger |
| ❌ Photographer only | Photographer jest partnerem biznesowym, nie primary Creatorem (v1.0+ dual role) |

## Value Proposition

### For RFG Creator

| Problem | RFG Solution |
|---------|-------------|
| „Instagram to nie portfolio — moje zdjęcia giną w feedzie" | Dedykowany portfolio z kategoryzacją (editorial, commercial, artistic) |
| „Dostaję fake casting offers na IG" | Verified Brand accounts (Sumsub KYC — doc 22); safe contact flows (doc 46) |
| „Nie umiem pitchować marek" | AI Brand Pitch tool (doc 41, IP-09) |
| „Nie wiem ile warta jest moja praca" | Rate guidelines + benchmark data (v0.5+) |
| „Musiszę mieć agencję żeby dostać dobre zlecenia" | Discovery system: Brands find Creators directly |

### For Brands (B2B — v0.5+)

| Problem | RFG Solution |
|---------|-------------|
| „Casting modeli to chaos (email, DM, agencje)" | Structured casting flow: brief → matching → contact |
| „Nie mam budżetu na agencję" | Direct-to-Creator: mniej intermediary = niższe koszty |
| „Potrzebuję weryfikacji modeli (wiek, prawa do wizerunku)" | Verification system (doc 46): L2 KYC + rights confirmation |

## Differentiation

### RFG vs. competitors

| Competitor | Co robią | Czego brakuje | RFG advantage |
|-----------|---------|-------------|--------------|
| **Instagram** | Social media; Creators post photos | Nie jest portfolio; algorithm ≠ professional discovery; brak safety dla modeli | Professional portfolio + verified brands + AI tools |
| **Model Mayhem** | Portfolio + casting (US-centric) | Dated UX; no AI; limited PL market; trust issues | Modern UX + Polish market + AI tools + safety |
| **Casting databases (e.g., e-Casting)** | Casting dla agencji | Pay-to-play; no social features; agency-gated | Free to join; social discovery; no middle-man |
| **OnlyFans** | Paywall content (often adult) | Stigma; ≠ portfolio; no brand collaboration | Professional positioning; zero NSFW; brand-safe |
| **TFP groups (Facebook)** | Trade-for-print photography | Unmoderated; scam-rich; zero verification | Verified Creators + safe contact + moderation |

### Positioning Statement

> **RFG jest profesjonalną platformą portfolio i discovery dla twórczyń wizualnych — łączy modele z weryfikowanymi markami w bezpiecznym, brand-safe środowisku z AI-powered narzędziami.**

## Premium Logic

### Why RFG can charge

| Argument | Opis |
|----------|------|
| **Portfolio is career tool** | Unlike social media, portfolio directly generates income (casting, brand deals) |
| **Safety has value** | Verified brands = protection from scam; worth paying for |
| **AI tools save money** | AI-generated brand pitch = no need for agency/copywriter |
| **Discovery = jobs** | Being found by brands = direct revenue for Creator |

### Revenue Streams (→ doc 47)

| Stream | Timing |
|--------|--------|
| Free tier (basic portfolio) | v0.5 |
| RFG Pro (premium portfolio + discovery boost + AI tools) | v0.5 |
| Brand matchmaking (B2B: brands pay to browse/contact) | v0.7 |
| Casting flow (B2B: structured casting with payment) | v1.0 |
| AI credits (shared with HHU ecosystem) | v0.5 |

## Brand Boundaries

### What RFG IS

| YES | Opis |
|-----|------|
| ✅ Professional portfolio platform | Creator shows best work |
| ✅ Discovery / matching | Brands find Creators |
| ✅ Safe collaboration space | Verified contacts, reporting |
| ✅ AI-assisted content tools | Bio, pitch, caption, cover |
| ✅ Premium subscriptions | Pro features for serious Creators |

### What RFG is NOT

| NO | Opis | Consequence |
|----|------|------------|
| ❌ Adult content platform | Zero NSFW policy (doc 27, 28) | Immediate account termination for violations |
| ❌ Dating app | No romantic matching features | No „swipe" mechanics ever |
| ❌ Modeling agency | RFG doesn't take booking commissions (v1.0: optional) | Not an agency; platform only |
| ❌ Stock photo marketplace | Creators own all rights; no licensing on platform | No rights transfer via RFG |
| ❌ General social network | Niche: visual Creators only | No expansion to „everyone" |

## Rekomendacja pozycjonowania

### RDZEŃ (core)

1. **Professional portfolio platform** — must work perfectly
2. **Verified brand matching** — trust = foundation
3. **Safety-first design** — every feature assessed for safety implications

### WSPARCIE (support)

4. AI tools for Creators (bio, pitch, content ideas)
5. Community features (light: follows, likes — not full social)
6. Analytics for Creators (who viewed portfolio, brand impressions)

### NIE ROBIMY (excluded)

7. ❌ Dating/matching mechanics
8. ❌ Adult content monetization
9. ❌ Full social network (feed-heavy, comment-heavy)
10. ❌ Modeling agency functions (booking, commission)
