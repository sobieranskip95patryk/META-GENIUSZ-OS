## Executive Summary

HHU startuje **wyłącznie w Polsce** (FDN-86). Rynek polski to idealne pole testowe: hip-hop jest gatunkiem #1 na streaming, scena jest aktywna ale fragmentaryczna, brak dedykowanej platformy. GTM strategy: **creator-led launch** — najpierw artyści (doc 36), potem ich fani, potem expansion organiczny. Zero performance marketing w Year 1 — growth przez content, partnerships i community.

---

## Rynek polski — kontekst

| Fakt | Źródło / szacunek |
|------|-------------------|
| Hip-hop = #1 gatunek muzyczny w PL (streaming) | Spotify PL Charts; ZPAV raporty |
| ~15M aktywnych użytkowników social media (18-34) | GUS / Gemius |
| Scena hip-hop PL: 500+ aktywnych artystów (professional to semi-pro) | Szacunek: label rosters + indie artists |
| Brak dedykowanej platformy hip-hop | Analiza rynku (doc 31) |
| Monetyzacja artystów PL = streaming (low payout) + koncerty + merch | Branżowe |
| Creator economy PL rośnie: Patronite (PL Patreon) ma ~5000 twórców | Patronite public data |
| PLN = primary currency; MRR reporting w EUR/USD | Finanse |

## Target Audience — warstwy

| Warstwa | Kto | Rozmiar | Prioritet GTM |
|---------|-----|---------|--------------|
| **Core** | Polscy raperzy, producenci, DJ-e | ~500 active | #1 — bez nich nie ma platformy |

| **Adjacent** | Managerowie, marki streetwear, media hip-hop | ~200+ organizacji | #4 — v0.5+ (B2B revenue) |

## Launch Strategy: 3 Phases

### Phase 1: Pre-Launch (T-4 do T-1 tygodnie)

**Cel**: Seeded content + Creator critical mass (≥50 Creators, ≥200 postów)

| Tydzień | Akcja | KPI |
|---------|-------|-----|
| T-4 | Finalizacja MVP (doc 33); staging environment ready | Build complete |
| T-4 | Pitch one-pager + demo access ready | Materiały done |
| T-3 | Outreach: 10 Ambasador candidates (doc 36) | ≥5 potwierdzeń |
| T-3 | Outreach: 40 Early Adopter candidates (doc 36) | ≥25 potwierdzeń |
| T-2 | Onboarding Ambasadorów: profiles, first posts, feedback | 5 active profiles |
| T-2 | Onboarding Early Adopters: konta, verification L1 | 25 active profiles |
| T-1 | Seeded content audit: ≥200 postów na platformie | Content check |
| T-1 | Cross-promo scheduling z Ambasadorami | Announcements queued |
| T-1 | Landing page live: hhu.pl (FDN-84) z waitlist | Landing live |

#### First Creator Clusters

| Cluster | Profil | Skąd brać |
|---------|--------|----------|
| **Trap/Drill PL** | Young artists, very active on IG/TikTok | DM outreach + manager contacts |
| **Boom Bap / Lyrical** | Established rappers, blog-era veterans | Founder's network + label contacts |
| **Producenci / Beatmakerzy** | Active on YouTube/BeatStars, need better promo | Reddit/FB groups + direct outreach |
| **Freestyle / Battle rap** | Competitive MCs, Klanzor/WBW community | Event contacts + social DM |
| **Polo/Disco polo-rap fusion** | Massive fan base, underserved online | Kontrowersyjny ale = audience; strategic pick |

> **Cel**: Diverse clusters = diverse content = szerokie pokrycie polskiego hip-hopu.

|-------|---------|---------------|

## Outreach Model

### 1:1 Creator Outreach (Founder-led)

| Element | Opis |
|---------|------|
| **Kto prowadzi** | Founder — 100% (solo dev constraint — FDN-90) |
| **Kanał** | DM (Instagram) → Call/Video → Onboarding link |
| **Pitch key points** | „Pierwsza platforma social stworzona dla hip-hopu. AI tools free. Twoi fani, Twój content, Twoje zarobki." |
| **Materials** | One-pager (PDF) + demo login + 3 screenshots |
| **Follow-up** | 48h po pierwszym DM; max 2 follow-ups; no spam |
| **Success metric** | 30% positive response rate on DMs |

### Pitch Template (One-Pager)

```
HIP HOP UNIVERSE — Platforma dla polskiego hip-hopu

🎤 Problem: Twoja muzyka ginie w feedach IG i TikTok, które nie są dla Ciebie.

🔥 Rozwiązanie: HHU — pierwsza platforma social stworzona dla hip-hopu.
   - Twój profil, Twoi fani, Twoje zasady
   - AI tools: bio, captions, promo assets — za darmo
   - Tips & subscriptions — zarabiaj bezpośrednio od fanów

🚀 Early Access: Dołącz do [5] artystów, którzy budują HHU od początku.
   - ✓ Free AI tools (unlimited, 12 miesięcy)
   - ✓ Featured placement na launch
   - ✓ Głos w rozwoju platformy

📱 Demo: [link]
📧 Kontakt: [Founder email]
```

## Activation Metrics

### Funnel

```
Awareness → Visit landing → Sign up → Complete profile → First post → Day 7 return → Day 30 return
```

### Targets

| Metryka | Launch (T1-2) | Growth (T3-12) | Alarm threshold |
|---------|---------------|----------------|-----------------|
| Landing page visits | 5,000 | 20,000 cumulative | <2,000 |
| Sign-up conversion | 15% | 12% | <8% |
| Profile completion | 60% | 70% | <40% |
| First post within 24h | 30% | 40% | <20% |
| D7 retention | 40% | 45% | <25% |
| D30 retention | 20% | 30% | <15% |
| DAU/MAU ratio | 20% | 25% | <15% |

### North Star Metric

> **Creator posts per week** — jeśli artyści publikują, fani przychodzą. Jeśli nie publikują, platforma umiera.

| Target | Wartość |
|--------|---------|
| Launch week | ≥100 posts/tydzień (from 50 Creators) |
| Month 1 end | ≥200 posts/tydzień |
| Month 3 end | ≥500 posts/tydzień |
