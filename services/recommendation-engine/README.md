# Recommendation Engine

Serwis odpowiadający za generowanie rekomendacji dla użytkowników.

## Features

- Collaborative filtering
- Content-based recommendations
- Hybrid recommendations
- Trending content
- Personalized feeds

## Machine Learning

- User similarity analysis
- Content embeddings
- A/B testing framework

## API Endpoints

- `GET /api/recommendations/feed` - Feed rekomendacji
- `GET /api/recommendations/similar/:id` - Podobna zawartość
- `GET /api/recommendations/trending` - Trendy

## Algorithm

- Matrix factorization
- k-NN
- Association rules
- Time decay factors
