# Moderation Service

Serwis odpowiadający za moderację treści i bezpieczeństwo.

## Features

- Content moderation
- Spam detection
- Inappropriate content filtering
- User reporting
- Moderation queue
- Appeals management

## Integration

- AI-powered content analysis
- Human moderation tools
- Automated actions (flag, remove, suspend)

## API Endpoints

- `POST /api/moderation/report` - Zgłoś zawartość
- `GET /api/moderation/queue` - Queue moderacji
- `POST /api/moderation/action` - Zastosuj akcję
- `POST /api/moderation/appeal` - Odwołanie od decyzji
