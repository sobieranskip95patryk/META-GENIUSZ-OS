# META-GENIUSZ OS

META-GENIUSZ OS is a unified AI-powered creator operating system combining:

- AI Core / LOGOS orchestration
- Hip Hop Universe
- Rocket Fuell Girls
- Creator Economy
- AI Generation
- Shared infrastructure and deployment architecture

## Modules

- `apps/web` - main platform entry
- `apps/hhu` - Hip Hop Universe
- `apps/rfg` - Rocket Fuell Girls
- `apps/ai-studio` - AI tools for creators
- `apps/admin` - admin and ops panel
- `apps/api` - backend API

## Packages

Shared packages for UI, database, auth, analytics, AI, agents, storage and payments.

## Vision

This repository is the master monorepo for the META-GENIUSZ OS ecosystem.
