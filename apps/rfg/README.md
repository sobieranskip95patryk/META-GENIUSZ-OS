# @meta-geniusz/rfg — Rocket Fuell Girls

Visual talent vertical dla modelek, muses, galerii i kampanii premium.

**Framework**: Next.js 16 (App Router) | **Pakiet**: `@meta-geniusz/rfg` | **Status**: 🔧 Scaffold

---

## Opis

`apps/rfg` to moduł premium META-GENIUSZ OS dedykowany visual talent — modelkom, artystkom wizualnym i kampaniom premium. Aktualnie w fazie scaffoldu — struktura gotowa do implementacji.

## Planowane funkcje (MVP)

- [ ] Strona główna RFG z galerią wyróżnionych talentów
- [ ] Profil modelki/muse (portfolio, bio, kontakt)
- [ ] Upload i zarządzanie galerią zdjęć
- [ ] System odkrywania talentów (discovery)
- [ ] Kampanie i brief system
- [ ] Integracja z `@meta-geniusz/ai-core` (rekomendacje)
